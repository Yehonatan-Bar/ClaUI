Detection Logic - Smarter Tracking

Implementing intelligent detection for each new achievement. Key capabilities:

  • File path extraction from tool inputs - tracking which files were touched and classifying them as frontend/backend
  • Error cycle counting - detecting when a developer persists through multiple error-fix-error cycles before succeeding
  • Edit precision analysis - detecting "surgical" fixes where 1-2 targeted edits resolve an error
  • Test-before-edit detection - comparing timestamps of first test run vs first edit to award test-first methodology
  • Daily streak tracking - comparing session dates across days to maintain consecutive day counts
  • Goal completion monitoring - checking if all session goals are met for the "All Goals Met" achievement
4

Cross-Session Intelligence

Building on the persisted profile counters to award long-term achievements:

  • Test Master tiers - Similar to Bug Slayer (I/II/III), rewarding cumulative test passes at 25/100/500
  • Centurion - Celebrating the 100th session, a milestone of dedication
  • Daily streaks - Tracking consecutive days and awarding at 3-day and 7-day milestones
  • Cumulative session time - Tracking total minutes spent coding with Claude
AI

AI Session Insights (Sonnet) AI-Powered

The headline feature: once per day, at session end, the system spawns a Claude Sonnet analysis of the session. This provides:

  • Session Quality Rating - Exceptional / Productive / Steady / Exploratory / Struggling
  • Insight Text - 1-2 sentence AI-generated observation about the session's coding patterns
  • Coding Pattern - Deep-dive / Breadth-first / Iterative / Planning-heavy / Test-driven
  • XP Bonus - 0-25 bonus XP awarded based on session quality assessment

The insight text and quality rating appear in the Session Recap card at session end. Rate-limited to once per calendar day to manage costs. The feature is best-effort - if the analysis fails or times out, the session recap shows normally without AI data.

Enhanced Session Recap

  • Files touched - Count of unique files modified during the session
  • Languages used - List of programming languages detected
  • AI insight - The Sonnet-generated observation (when available)
  • Session quality - Visual quality badge from AI analysis
  • Coding pattern - What kind of work pattern was detected
  • AI XP bonus - Extra XP awarded by the AI assessment

Translations & New Goals

Full Hebrew and English translations for all 18 new achievements, 3 new session goals, and the enhanced recap labels. Updated info modal to reflect 15 levels and AI insights.

3 new session goal templates: File Hopper (touch 8 files), Error Free (3 error-free results), Language Sampler (2+ languages).

Summary

This enhancement transforms the achievement system from a basic 12-achievement tracker into a comprehensive 30-achievement system with AI-powered insights. The system will track deeper coding patterns (file operations, frontend/backend work, error persistence, daily consistency), offer 15 progression levels, and provide daily AI-generated session analysis via Sonnet. All changes maintain backward compatibility with existing profiles and are fully translated to Hebrew.
