ClaUi - שיחה מרובת משתתפים

מדריך למשתמש: איך ליצור או להצטרף לשיחה משותפת עם סוכני AI

1
משתמש ראשון - יוצר הסשן
מי שפותח את השיחה ומזמין אחרים להצטרף
דרישות מקדימות
  • תוסף ClaUi מותקן ב-VS Code
  • גישה לשרת התיאום (כתובת URL וטוקן אימות מהמנהל)
1
הגדרת חיבור לשרת (פעם אחת בלבד)
פתח את הגדרות VS Code:
Ctrl+Shift+P ← Preferences: Open Settings (JSON)
הוסף את השורות הבאות:
הגדרה ערך
claudeMirror.multiParticipant.serverUrl כתובת השרת שקיבלת (למשל wss://server.com/mp)
claudeMirror.multiParticipant.authToken הטוקן הסודי שקיבלת מהמנהל
claudeMirror.multiParticipant.defaultHumanName השם שלך (למשל "יוני")
זו הגדרה חד-פעמית. אחרי שהזנת את הפרטים, הם יישמרו ולא תצטרך להזין אותם שוב.
2
יצירת סשן חדש
Ctrl+Shift+P ← ClaUi: Create Multi-Participant Session
3
הזנת פרטים
VS Code יבקש ממך למלא:
  • השם שלך - איך תופיע בשיחה (למשל "יוני")
  • שם הסוכן - איך ייקרא הסוכן שלך (למשל "Claude")
  • שם הסשן - נושא השיחה (למשל "סקירת קוד", "תכנון פרויקט")
  • מספר סשן - מספר ייחודי לזיהוי (למשל 42)
  • סיסמה - אופציונלי, להגנה על הסשן
4
הסשן נפתח
ייפתח טאב חדש ב-VS Code עם שם הסשן. בפינה תראה את מספר הסשן (למשל #42) ותג חיבור ירוק.
שתף עם המשתתפים האחרים:
שלח להם את מספר הסשן (למשל 42) ואת הסיסמה (אם הגדרת). הם צריכים רק את המספר כדי להצטרף.
2
משתמש שני - מצטרף לסשן קיים
מי שקיבל מספר סשן ורוצה להצטרף לשיחה
מה צריך לפני
  • תוסף ClaUi מותקן ב-VS Code
  • כתובת השרת והטוקן (אותם פרטים כמו למשתמש הראשון)
  • מספר הסשן שקיבלת מיוצר הסשן
  • סיסמת הסשן (אם הוגדרה)
1
הגדרת חיבור לשרת (פעם אחת בלבד)
בדיוק כמו אצל המשתמש הראשון - פתח הגדרות VS Code והזן את כתובת השרת, הטוקן, והשם שלך. ראה את הטבלה בחלק של המשתמש הראשון.
2
הצטרפות לסשן
Ctrl+Shift+P ← ClaUi: Join Multi-Participant Session
3
הזנת פרטים
VS Code יבקש ממך:
  • השם שלך - איך תופיע בשיחה
  • שם הסוכן - השם שייתן לסוכן שלך
  • מספר סשן - המספר שקיבלת מיוצר הסשן
  • סיסמה - אם הסשן מוגן בסיסמה
4
הסשן נפתח
ייפתח טאב עם שם הסשן. תראה את כל ההודעות הקודמות ואת רשימת המשתתפים בצד.
אם קיבלת הודעת שגיאה "No session found" - ודא שהמספר נכון ושיוצר הסשן כבר פתח אותו. אם קיבלת "Invalid session password" - בדוק את הסיסמה עם יוצר הסשן.
