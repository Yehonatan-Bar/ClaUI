Usage Tips - טיפים לשימוש בממשק

מה זה?
פיצ'ר חדש שמציג למשתמש טיפים על יכולות נסתרות בממשק ClaUi. הטיפים מופיעים 5 פעמים ביום, בעיצוב ייחודי שמבדיל אותם בצורה ברורה מהודעות הצ'אט.

למה זה נחוץ?

ל-ClaUi יש 32 קטגוריות של פיצ'רים עם למעלה מ-160 יכולות שונות, אבל רוב המשתמשים מגלים רק חלק קטן מהם. ה-README מתעד רק כ-30% מהפיצ'רים. הטיפים עוזרים למשתמשים לגלות פיצ'רים שהם לא מכירים.

איך זה נראה?

i ?DID YOU KNOW
File Mention (@)
Type @ in the input box to search your workspace files. Select a file to insert its path into your message.
Got it 2/5 today

עיצוב ייחודי: הטיפים מוצגים עם פס צהוב-זהב בצד, רקע גרדיאנט עדין, ואייקון "i" עגול. זה נראה שונה לחלוטין מהודעות הצ'אט, כדי שהמשתמש יבין מיד שזה טיפ מערכת ולא חלק מהשיחה עם Claude.

איך זה עובד?

1
הטיפ הראשון של היום מופיע כ-2 דקות אחרי שמתחיל סשן חדש
2
טיפים נוספים מופיעים כל 45 דקות של שימוש פעיל
3
המשתמש לוחץ "Got it" כדי לסגור את הטיפ
4
מוניט "X/5 today" מראה כמה טיפים נותרו להיום
5
בחצות הספירה מתאפסת, והסבב ממשיך מהטיפ הבא ברשימה

מתי הטיפים לא מופיעים?

מצבסיבה
Claude עסוקלא מפריעים בזמן עבודה
אישור תוכנית (Plan Approval)לא מוסיפים עומס כשצריך החלטה
סיכום סשן מוצגלא מתחרים עם UI אחר
הגענו ל-5 טיפים היוםמכסה יומית
הפיצ'ר כבוי בהגדרותהמשתמש בחר לא לראות טיפים

סיכום טכני

קבצים חדשים: 2
קבצים שמשתנים: 7
מיקום בממשק: מתחת לשורת הקלט, מעל הסטטוס בר
אחסון: localStorage בצד ה-webview (ללא עלות API)
עלות: אפס -- הטיפים הם טקסט סטטי, אין קריאות API
כמות טיפים: ~50, מכסים את כל 32 הקטגוריות
סבב מלא: ~10 ימים (50 טיפים / 5 ליום)
