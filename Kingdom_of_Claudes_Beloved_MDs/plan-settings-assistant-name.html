תוכנית פיתוח | Claude Code Mirror

מה נבנה?

כפתור הגדרות חדש בתחתית הצ'אט, שמאפשר למשתמש לשנות את השם "Assistant" לכל שם שירצה.

למשל: במקום "Assistant" יופיע "Claude", "Bot", או כל שם מותאם אישית.

איך זה עובד?

Settings Button → Text Input → Save to VS Code → Update Chat UI

השם נשמר בהגדרות VS Code, כך שהוא נשאר גם אחרי סגירה ופתיחה מחדש של התוכנה.

שלבי הביצוע

1
הגדרת אפשרות חדשה
הוספת שדה הגדרה חדש בשם "שם תצוגה לעוזר" עם ברירת מחדל "Assistant"
2
חיבור צד השרת
התוסף קורא את ההגדרה ושולח אותה לממשק הצ'אט. כשהמשתמש משנה את השם, זה נשמר בהגדרות.
3
בניית כפתור ההגדרות
כפתור גלגל שיניים בשורת הסטטוס. לחיצה פותחת פאנל קטן עם שדה טקסט לעריכת השם.
4
עדכון הצ'אט
כל מקום שמציג "Assistant" (בועת הודעה, הודעה בזמן כתיבה) יציג את השם המותאם.

היקף השינוי

9 קבצים קיימים + 1 קובץ חדש

רכיב חדש אחד (SettingsPanel) + עדכון קבצים קיימים בלבד. עוקב אחרי אותו תבנית שכבר קיימת בפרויקט.

בדיקות

  • כפתור ההגדרות מופיע בשורת הסטטוס
  • לחיצה פותחת פאנל עם שדה שם
  • שינוי השם מתעדכן מיידית בצ'אט
  • השם נשמר בין הפעלות
  • אפשר לשנות את השם גם דרך הגדרות VS Code
  • ריקון השדה מחזיר לברירת המחדל "Assistant"
