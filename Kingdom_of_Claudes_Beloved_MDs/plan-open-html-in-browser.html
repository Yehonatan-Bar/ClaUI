תוכנית: פתיחת מסמכי תוכנית מתוך הממשק

Claude Code Mirror - תוספת פיצ'ר חדש

מה הבעיה?

בזמן תכנון משימות, התוסף יוצר מסמכי תוכנית בפורמט HTML. כרגע אין דרך פשוטה לפתוח אותם מתוך הממשק של התוסף - צריך לחפש את הקובץ ידנית בתיקיות.

מה הפתרון?

הוספת כפתור "Plans" בשורת הסטטוס של הצ'אט. לחיצה עליו תפתח את מסמך התוכנית בדפדפן.

איך זה עובד?

1
המשתמש לוחץ על כפתור "Plans" בשורת הסטטוס
2
המערכת סורקת את תיקיית המסמכים ומחפשת קבצי HTML
3
אם יש קובץ אחד - הוא נפתח ישירות בדפדפן
4
אם יש מספר קבצים - מוצגת רשימת בחירה (ממוינת לפי תאריך, החדש ביותר למעלה)
5
הקובץ שנבחר נפתח בדפדפן ברירת המחדל של המשתמש

מיקום הכפתור

הכפתור ימוקם בשורת הסטטוס התחתונה של הצ'אט, ליד כפתור "History" הקיים. העיצוב יהיה זהה לכפתורים הקיימים לשמירה על עקביות.

בנוסף, הפקודה תהיה זמינה גם דרך לוח הפקודות של VS Code (Ctrl+Shift+P -> "Claude Mirror: Open Plan Document").

מה משתנה?

רכיב שינוי היקף
שורת הסטטוס כפתור "Plans" חדש קטן
פרוטוקול הודעות הודעה חדשה: openPlanDocs קטן
פקודות התוסף פקודה חדשה לפתיחת מסמכים בינוני
עיצוב סגנון CSS לכפתור החדש קטן

סיכום

שינוי קטן וממוקד ב-6 קבצים, שמוסיף גישה נוחה למסמכי תוכנית ישירות מתוך ממשק הצ'אט. השינוי עוקב אחרי דפוסים קיימים בקוד ולא מוסיף מורכבות חדשה.
