כפתור דחיפה לגיט

תוכנית הוספת כפתור Git Push לממשק עם תמיכה בקונפיגורציה

מה מוסיפים?

כפתור "Git" באזור ההקלדה שמבצע את כל תהליך הדחיפה לגיט בלחיצה אחת:

  • מוסיף את כל השינויים
  • יוצר הגבה (commit) עם שם הסשן כהודעת הגבה
  • דוחף לשרת המרוחק

בנוסף - פאנל קונפיגורציה שמאפשר לבקש מקלוד להגדיר או לשנות את ההגדרות.

איך זה עובד?

מצב 1: הקונפיגורציה לא הוגדרה

1

המשתמש לוחץ על כפתור "Git"

2

נפתח פאנל קונפיגורציה עם שדה קלט

3

המשתמש כותב הוראה (למשל: "הגדר דחיפה לגיט לפרויקט הזה")

4

ההוראה נשלחת לקלוד שמגדיר את ההגדרות אוטומטית

מצב 2: הקונפיגורציה הוגדרה

1

המשתמש לוחץ על כפתור "Git"

2

הסקריפט רץ מיד - מוסיף, מגיב, ודוחף לגיט

3

מופיעה הודעת הצלחה או שגיאה (נעלמת אחרי 5 שניות)

כפתור הגדרות (תמיד זמין)

כפתור גלגל שיניים קטן ליד כפתור Git - תמיד נגיש, גם כשהקונפיגורציה כבר מוגדרת.

מאפשר לבקש מקלוד לשנות הגדרות קיימות (למשל: "שנה את תבנית ההגבה", "עדכן את הנתיב לסקריפט").

ההגדרות שנוספות

האם מופעל מציין אם הכפתור מוכן לשימוש (ברירת מחדל: לא)
נתיב הסקריפט נתיב לסקריפט הדחיפה (ברירת מחדל: scripts/git-push.ps1)
תבנית הודעת הגבה תבנית להודעת הגבה, עם {sessionName} כשם הסשן

קבצים מושפעים

חדש GitPushPanel רכיב הפאנל עם שדה הקלט והסטטוס
שינוי InputArea הוספת כפתור Git + גלגל שיניים + הודעות מצב
שינוי MessageHandler טיפול בהרצת הסקריפט ושליחת הוראות קונפיגורציה
שינוי הגדרות הודעות 5 סוגי הודעות חדשים לתקשורת
שינוי מנוע המצב שמירת הגדרות, תוצאות, ומצב הפאנל
שינוי הגדרות התוסף 3 שדות הגדרה חדשים ב-package.json
שינוי עיצוב סגנונות לכפתור, פאנל, והודעות מצב

סיכום

פשוט לשימוש: לחיצה אחת על "Git" דוחפת הכל לגיט.

חכם: אם עוד לא הוגדר - מכוון את המשתמש להגדרה דרך קלוד.

גמיש: תמיד אפשר לשנות הגדרות דרך פאנל ההגדרות.
