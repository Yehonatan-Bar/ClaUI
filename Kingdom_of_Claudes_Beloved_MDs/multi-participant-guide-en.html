ClaUi - Multi-Participant Sessions

User guide: how to create or join a shared session with AI agents

1
First User - Session Creator
The person who opens the session and invites others to join
Prerequisites
  • ClaUi extension installed in VS Code
  • Access to the coordination server (URL and auth token from your admin)
1
Configure Server Connection (one-time setup)
Open VS Code settings:
Ctrl+Shift+P → Preferences: Open Settings (JSON)
Add the following entries:
Setting Value
claudeMirror.multiParticipant.serverUrl The server URL you received (e.g. wss://server.com/mp)
claudeMirror.multiParticipant.authToken The shared secret token from your admin
claudeMirror.multiParticipant.defaultHumanName Your display name (e.g. "Alice")
This is a one-time setup. Once entered, these settings are saved and you won't need to enter them again.
2
Create a New Session
Ctrl+Shift+P → ClaUi: Create Multi-Participant Session
3
Fill In the Details
VS Code will prompt you for:
  • Your name - How you'll appear in the chat (e.g. "Alice")
  • Agent name - What your AI agent will be called (e.g. "Claude")
  • Session name - Topic of the session (e.g. "Code Review", "Sprint Planning")
  • Session number - A unique number to identify this session (e.g. 42)
  • Password - Optional, to protect the session from uninvited participants
4
Session Opens
A new tab opens in VS Code with the session name as the title. You'll see the session number as a badge (e.g. #42) and a green connection indicator.
Share with other participants:
Send them the session number (e.g. 42) and the password (if you set one). They only need the number to join.
2
Second User - Joining an Existing Session
Someone who received a session number and wants to join
What You Need
  • ClaUi extension installed in VS Code
  • Server URL and auth token (same details as the first user)
  • The session number from the session creator
  • The session password (if one was set)
1
Configure Server Connection (one-time setup)
Same as the first user - open VS Code settings and enter the server URL, auth token, and your name. See the settings table in the section above.
2
Join the Session
Ctrl+Shift+P → ClaUi: Join Multi-Participant Session
3
Fill In the Details
VS Code will prompt you for:
  • Your name - How you'll appear in the chat
  • Agent name - What your AI agent will be called
  • Session number - The number you received from the session creator
  • Password - If the session is password-protected
4
Session Opens
A tab opens with the session name. You'll see all previous messages and the list of participants in the sidebar.
If you see "No session found" - double-check the session number and make sure the creator has already opened the session. If you see "Invalid session password" - verify the password with the session creator.

How to Send Messages

Once everyone is connected, all participants see messages in real time.

→
Regular message: Just type text - all humans in the session will see it.
Agents won't receive the message unless you address them by name.
→
Address an agent: Start with the agent's name.
Example: Claude check this file - sends a prompt to the Claude agent.
→
Shorthand: Use the first letter + colon.
Example: C: check this file - shortcut to address Claude.
