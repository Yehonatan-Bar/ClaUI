תוכנית תמיכת MCP בתוסף ClaUI

מה זה MCP?

Model Context Protocol הוא תקן פתוח שמחבר את Claude לכלים חיצוניים -- GitHub, Sentry, מסדי נתונים, Slack, ועוד. זה מה שהופך את Claude מ"עוזר קוד" ל"סוכן פיתוח" שמסוגל לעבוד מול מערכות אמיתיות.

כיום ClaUI מציג רק את שמות השרתים המחוברים, בלי סטטוס, בלי ניהול, ובלי שום אפשרות הגדרה מתוך הממשק.

מה אנחנו בונים

פאנל ניהול MCP מלא שמאפשר למשתמש לראות, להוסיף, להסיר ולנטר שרתי MCP ישירות מתוך ClaUI -- בלי צורך בשורת פקודה.

לוח מצב חי

סטטוס חיבור בזמן אמת לכל שרת -- מחובר, צריך אימות, שגיאה, או מנותק.

מיפוי כלים לשרתים

תצוגה ברורה של איזה כלים שייכים לאיזה שרת MCP, מקובצים ומסוננים.

הוספה והסרה קלה

טופס נוח להוספת שרתים חדשים, עם תמיכה בכל סוגי התחבורה (stdio, HTTP, SSE).

שרתים פופולריים בלחיצה

גלריית שרתים מוכרים (GitHub, Playwright, Sentry ועוד) -- לחיצה אחת למילוי אוטומטי של ההגדרות.

איך זה ייראה

אינדיקטור בשורת המצב

כפתור קטן בשורת המצב התחתונה שמראה כמה שרתים מחוברים מתוך הסך הכולל. לחיצה פותחת את הפאנל המלא.

סטטוס שרתים

GitHub -- מחובר Codex -- מחובר Gmail -- צריך אימות Sentry -- שגיאה Slack -- מנותק

כרטיס שרת

כל שרת מוצג ככרטיס עם: שם, סטטוס חיבור (עיגול צבעוני), סוג חיבור (stdio/HTTP/SSE), היקף (משתמש/פרויקט/מקומי), ומספר הכלים הזמינים.

לחיצה על כרטיס מרחיבה את רשימת הכלים הספציפיים של אותו שרת.

שלבי פיתוח

שלב 1 -- גרסה ראשונה

המטרה: תצוגת סטטוס עשירה + ניהול בסיסי של שרתים

  • העשרת נתוני ההתחלה -- שמירת סטטוס חיבור, רשימת כלים לכל שרת, סוג תחבורה
  • פאנל MCP מלא עם שלושה טאבים: שרתים, כלים, הוספת שרת
  • כרטיסי שרת עם סטטוס חיבור צבעוני ופרטים
  • מיפוי כלים-לשרתים -- תצוגה מקובצת לפי שרת
  • טופס הוספת שרת עם תמיכה בכל סוגי התחבורה
  • גלריית שרתים פופולריים להוספה מהירה
  • הסרת שרת עם אישור
  • אינדיקטור MCP בשורת המצב
  • עדכון תצוגת הדשבורד הקיימת עם צבעי סטטוס
שלב 2 -- ניטור מתקדם

המטרה: מעקב בריאות שרתים והתראות

  • מעקב פעילות כלים בזמן אמת -- זמן אחרון של שימוש בכל שרת
  • מעקב אחר שינויים בקבצי הגדרות -- רענון אוטומטי של הממשק
  • פרטי שגיאה מורחבים עם כפתור "אמת מחדש" לשרתים שצריכים אימות
  • לוג חיבורים לכל שרת
שלב 3 -- יכולות מתקדמות

המטרה: עריכה, בדיקה, ושיתוף הגדרות

  • עריכת הגדרות שרת ישירות מהממשק (פקודה, כתובת, משתני סביבה)
  • כפתור "בדוק חיבור" -- בדיקת תקינות לכל שרת
  • אשף יצירת קובץ הגדרות פרויקטי (.mcp.json)
  • ייצוא וייבוא הגדרות MCP לשיתוף בין מפתחים
  • טאב MCP בדשבורד עם סטטיסטיקות פעילות

החלטות טכניות עיקריות

נושא החלטה סיבה
ניהול הגדרות שימוש בפקודות CLI של Claude בטוח יותר, שומר על תאימות, מונע קונפליקטים
תצוגת נתונים קריאת קובץ ההגדרות ישירות נתונים מובנים, מהיר, אמין
סוג ממשק פאנל מסך מלא (כמו Dashboard/Teams) עקביות עם דפוסי עיצוב קיימים בתוסף
מקור סטטוס אירוע התחלה + מעקב כלים ללא צורך בתשאול מתמשך, חסכוני

סיכונים ומענה

סיכון מענה
פקודות CLI איטיות (עד 10 שניות לשרתי HTTP) טעינה א-סינכרונית עם אנימציה, timeout של 15 שניות
שינוי בפורמט קובץ ההגדרות קריאה דפנסיבית עם ברירות מחדל
בעיות ב-Windows עם npx תבניות שרתים פופולריים כוללות עטיפת cmd /c

סיכום

התוכנית הזו מביאה תמיכת MCP מלאה ל-ClaUI בגישה מדורגת. שלב 1 נותן למשתמשים שליטה מלאה על שרתי MCP ישירות מהממשק -- ללא צורך בשורת פקודה. שלבים 2 ו-3 מוסיפים ניטור מתקדם ויכולות ניהול עשירות.

הפיתוח מבוסס על דפוסים מוכחים שכבר קיימים בתוסף (כמו פאנל Teams ו-Dashboard), ומשתמש ב-CLI של Claude לכל הפעולות -- מה שמבטיח תאימות ובטיחות.
