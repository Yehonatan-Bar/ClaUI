גילוי כל השיחות מהמערכת

פיצ'ר חדש שמאפשר לגלות ולחזור לכל שיחה שהתקיימה עם Claude Code - גם לפני התקנת התוסף

הבעיה

כרגע, ההיסטוריה של ClaUi מציגה רק שיחות שנפתחו דרך התוסף עצמו. שיחות שנוהלו ישירות מהטרמינל (Claude Code CLI) לפני ההתקנה - לא נראות.

הפתרון: Claude Code שומר את כל השיחות כקבצים בתיקייה קבועה במחשב. נסרוק את התיקייה ונציג את כל השיחות - כולל אלו מלפני ההתקנה.
