ClaUi - שיתוף הישגים בין מפתחים

הפיכת מערכת ההישגים של ClaUi לפלטפורמה חברתית שבה מפתחים יכולים לשתף, להשוות ולהתהדר בהישגיהם

הרעיון בקצרה

מפתחים שמשתמשים ב-ClaUi צוברים הישגים (30 הישגים, 7 קטגוריות, 15 רמות). כרגע - הכל מקומי ופרטי. התוכנית הזו הופכת את ההישגים לפומביים: מפתח יוכל לפרסם את ההישגים שלו ב-GitHub, לצפות בהישגים של מפתחים אחרים מתוך הממשק, ולהציג תגי הישגים בפרופיל GitHub שלו.

למה דווקא GitHub?

כל מפתח כבר יש לו חשבון GitHub. VS Code כולל חיבור מובנה ל-GitHub (בלי תוספים חיצוניים). GitHub Gist מספק אחסון JSON חינמי ופומבי שכל אחד יכול לקרוא. הדפוס הזה מוכח - כלים כמו github-readme-stats (מעל 70,000 כוכבים) עובדים באותו עיקרון. לא צריך שרת בכלל - הכל עובד ישירות מול GitHub.

1 חיבור GitHub וסנכרון אוטומטי

השלב הבסיסי - חיבור חשבון GitHub ופרסום הישגים אוטומטי.

  • חיבור GitHub: המפתח לוחץ "Connect GitHub" בממשק ההישגים. VS Code מבקש הרשאת גישה (מובנית, בלי ספריות חיצוניות).
  • פרסום אוטומטי: בסיום כל סשן עבודה, ההישגים המעודכנים מתפרסמים כקובץ JSON ב-GitHub Gist (פומבי, חינמי).
  • גילוי מפתחים: כשמחפשים מפתח לפי שם המשתמש שלו ב-GitHub, המערכת מוצאת אוטומטית את ה-Gist של ההישגים שלו.
  • עמידות: אם אין חיבור לאינטרנט, הסנכרון נכשל בשקט - שום דבר לא נשבר. הנתונים המקומיים תמיד המקור האמיתי.

ברירת מחדל: כבוי הגדרה חדשה

2 קהילה - חברים והשוואה

ממשק חברתי מלא בתוך ClaUi לצפייה והשוואת הישגים.

  • הוספת חברים: הזנת שם משתמש GitHub - המערכת מוצאת את ההישגים שלו ומוסיפה לרשימת חברים.
  • רשימת חברים: תצוגת כל החברים עם תמונת פרופיל, רמה, XP, מספר הישגים.
  • השוואה מול חבר: תצוגה זה-לצד-זה - רמה מול רמה, הישג מול הישג, סטטיסטיקות מול סטטיסטיקות. אפשר לראות בדיוק אילו הישגים יש לך ואילו לחבר.
  • כפתור "קהילה": כפתור חדש בראש פאנל ההישגים שפותח את לוח הקהילה.

פאנל חדש תמיכה בעברית ואנגלית

3 כרטיס הישגים לפרופיל GitHub

יצירת תגים ו"כרטיס הישגים" שאפשר להטמיע בפרופיל GitHub.

  • תגי Shields.io דינמיים: תגים שמושכים נתונים ישירות מה-Gist ומתעדכנים אוטומטית - רמה, XP, מספר הישגים. ניתן להדביק ב-README של הפרופיל.
  • כרטיס Markdown: טבלה מעוצבת עם כל הסטטיסטיקות - רמה, XP, הישגים אחרונים, תיקוני באגים, בדיקות שעברו, רצף ימים.
  • כפתור "שיתוף": לחיצה אחת מעתיקה את ה-Markdown ללוח - פשוט להדביק בפרופיל.
  • תצוגה מקדימה: לפני ההעתקה, אפשר לראות איך הכרטיס ייראה.

סיכום

הפיצ'ר הזה הופך את מערכת ההישגים של ClaUi מכלי מקומי לפלטפורמה חברתית בינלאומית. מפתחים יוכלו להתחרות, להשוות ולהתהדר - הכל בתוך הכלי שהם כבר משתמשים בו. הארכיטקטורה פשוטה (ללא שרת), ללא עלות, ומבוססת על כלים שכל מפתח כבר מכיר.
