# Archive

This folder contains historical documents that are no longer current-state documentation.

## Contents

### `plans/`
Implementation plans, initial designs, and external reference documents. These describe what was planned or how features were initially designed, not the current implementation.

### `html-content/`
HTML exports of plans, executive summaries, and presentations. These are artifacts of planning discussions, not implementation documentation.

### `duplicate-folders/`
Duplicate folder trees (`New folder/`, `New folder - Copy/`) that contained copies of documents from the parent directory. Moved here to prevent confusion with canonical documents.

## Usage

These documents have historical and reference value but should NOT be treated as source of truth for the current codebase. For current implementation documentation, refer to the canonical `.md` files in the parent `Kingdom_of_Claudes_Beloved_MDs/` directory and `TECHNICAL.md` in the project root.
