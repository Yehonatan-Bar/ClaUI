תוכנית פיתוח: חיבור מרחוק דרך Happy Coder

הוספת ספק חדש לתוסף ClaUi שמאפשר שליטה מרחוק בסשנים של Claude דרך שרת Happy Coder

מה זה בכלל?

כיום, ClaUi מריץ את Claude באופן מקומי על המחשב. התוכנית הזו מוסיפה אפשרות שלישית: חיבור לסשן Claude שרץ על מחשב אחר, דרך שרת ממסר (relay) של Happy Coder. כל התקשורת מוצפנת מקצה לקצה.

מה המשתמש יראה?

  • כפתור חדש "Happy" בבורר הספקים (ליד Claude ו-Codex)
  • בחירת "Happy" פותחת חלון שבו אפשר ליצור סשן חדש או להתחבר לסשן קיים
  • ממשק הצ'אט נשאר זהה לחלוטין -- הודעות, כלים, סטטוס -- הכל עובד אותו דבר
  • הגדרות חדשות: כתובת שרת, מצב הרשאות, חיבור אוטומטי

ארכיטקטורה

הפתרון עוקב אחרי הדפוס הקיים בדיוק -- כמו שנוספה תמיכה ב-Codex כספק שני:

TabManager (מנהל הטאבים) | +----------+-----------+ | | | Claude Codex Happy (חדש) (מקומי) (מקומי) (מרחוק) | | | Process Process Connection Manager Manager Manager (תהליך) (תהליך) (Socket.IO) | | | +----------+-----------+ | Webview (React) ללא שינויים!

הגנת הרשאות (מ-happy-patch)

אחד הלקחים החשובים מ-happy-patch: כשמשתמש מגדיר מצב "הרשאות מלאות", צריך להגן על ההגדרה הזו. הודעות נכנסות מהשרת יכולות לנסות להחזיר את ההרשאות ל"רגיל" -- המערכת תחסום ירידה כזו אוטומטית.
