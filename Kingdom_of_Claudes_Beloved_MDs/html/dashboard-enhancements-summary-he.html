1. ניתוח סמנטי - מופעל כברירת מחדל

הניתוח הסמנטי שמזהה מצב רוח, סוג משימה, תוצאה, ובאגים חוזרים מופעל עכשיו אוטומטית מרגע ההתקנה.

קודם היה צריך להפעיל ידנית דרך ההגדרות - עכשיו זה עובד מייד. אפשר לכבות בכל רגע דרך כפתור ההגדרות (גלגל השיניים ליד Vitals) או דרך הגדרות VS Code.

מה שונה: ברירת מחדל בארבעה מקומות (הגדרת התוסף, מנתח הסמנטי, מנהל ההודעות, חנות המצב).

2. דף הבית (Overview) - יותר נתונים, בלי טוקנים

כרטיסי מטריקות חדשים

3 כרטיסים שקשורים לטוקנים הוסרו והוחלפו:

  • סך שימושי כלים - כמה פעמים השתמש קלוד בכלים בסה"כ
  • הכלי הנפוץ ביותר - שם הכלי + מספר הפעמים
  • פקודות Shell - כמה פקודות Bash הורצו

נשארו: עלות כוללת, מספר תורות, עלות ממוצעת, שיעור שגיאות, משך ממוצע.

גרפים חדשים

דף הבית מציג עכשיו 4 גרפים במקום 2:

  • עלות לתור - כמו קודם (מצטבר + לתור)
  • משך זמן לתור - חדש - צבעוני לפי קטגוריה (כתיבה, מחקר, פקודה, שגיאה, דיון)
  • תדירות כלים - חדש - 15 הכלים הנפוצים ביותר בגרף אופקי
  • התפלגות קטגוריות - חדש - תרשים דונאט עם אחוזים

פס מצב רוח והתראת תסכול נשארים כמו שהיו.

3. טאב חדש: בודק הקשר (Context)

טאב שישי חדש בדשבורד שמראה את כל התקשורת עם מודל השפה.

מטא-דאטה של הסשן

מוצג בראש הטאב כרטיס עם:

  • מזהה סשן
  • שם המודל הפעיל (צבעוני)
  • תיקיית עבודה
  • שרתי MCP מחוברים (בתגיות סגולות)
  • רשימת כלים זמינים (כל הכלים שהמודל יכול להשתמש בהם)

מציג שיחות מלא

כל הודעה ניתנת להרחבה בלחיצה. כשמורחבת, מציגה:

  • טקסט - תוכן הטקסט של ההודעה
  • קריאת כלי - שם הכלי + קלט מלא בפורמט JSON
  • תוצאת כלי - פלט הכלי כולל סטטוס שגיאה
  • תמונות מסומנות בהתאם

יכולות סינון וחיפוש

  • סינון לפי תפקיד: הכל / משתמש / עוזר
  • חיפוש טקסט חופשי בתוכן הודעות, שמות כלים, קלט כלים
  • כפתורי "הרחב הכל" / "כווץ הכל"
  • סרגל סטטיסטיקה: מספר הודעות, בלוקי תוכן, קריאות כלים

הערה חשובה: פרומפט מערכת

פרומפט המערכת (System Prompt) לא זמין דרך פרוטוקול ה-CLI של קלוד. זו מגבלה של הפרוטוקול עצמו.

הטאב מציג את כל מה שכן זמין: מטא-דאטה מאירוע האתחול (כלים, מודל, תיקייה, שרתי MCP) ואת כל הודעות השיחה עם כל בלוקי התוכן.

לעומת כלי כמו llm-visuals שעובד כפרוקסי הפוך ויכול ליירט את הפרומפט, ClaUi עובד מעל CLI קיים ורואה רק את מה שהפרוטוקול חושף.
