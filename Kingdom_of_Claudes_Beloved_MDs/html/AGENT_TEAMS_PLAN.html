מה זה Agent Teams? סקירת הפיצ'ר

ריבוי סוכנים שעובדים במקביל

Agent Teams מאפשר להריץ מספר סשנים של Claude Code במקביל, כאשר סשן אחד משמש כ-מוביל הצוות (Team Lead) והשאר הם חברי צוות (Teammates) שעובדים באופן עצמאי. כל סוכן מקבל הקשר עצמאי, יכול לשלוח הודעות ישירות לסוכנים אחרים, ולעבוד על משימות מתוך רשימה משותפת.

מוביל צוות

הסשן הראשי שיוצר את הצוות, מחלק משימות, ומתאם בין הסוכנים. רואה את כל הפעילות ומנהל את מחזור החיים.

חברי צוות

סוכנים עצמאיים שמקבלים משימות, עובדים על קוד, ומדווחים תוצאות. כל אחד עם חלון הקשר נפרד ויכולת עבודה מלאה.

תקשורת ישירה

הסוכנים יכולים לשלוח הודעות ישירות אחד לשני - לא רק דרך המוביל. שיתוף ממצאים ותיאום ללא צוואר בקבוק.

⚠

הבעיה למה צריך את הפיצ'ר הזה

הפעילות של הצוות בלתי נראית ב-VS Code

כרגע, כשמפעילים Agent Teams ב-ClaUi, המשתמש רואה רק את השיחה של מוביל הצוות. אין שום ויזואליזציה של מה שקורה ברקע - אין מעקב אחרי סוכנים, אין תצוגת משימות, אין הודעות בין-סוכנים. המשתמש עובד "בחושך" לגבי רוב הפעילות של הצוות שלו.

שליטה בזרימה BONUS - פעולות ישירות מהממשק

לא רק צפייה - גם שליטה

המשתמש לא רק יראה מה קורה, אלא גם יוכל לפעול ישירות מהממשק: לשלוח הודעות לסוכנים, ליצור משימות חדשות, לעדכן סטטוס, ואפילו לכבות סוכנים - הכל בלחיצת כפתור, בלי לכתוב פקודות.

✉

שליחת הודעות

שליחת הודעה ישירה לסוכן ספציפי או שידור לכל הצוות - ישירות מלשונית ההודעות

➕

יצירת משימות

הוספת משימות חדשות ללוח הקנבן, כולל תיאור, עדיפות ותלויות

✏

עדכון משימות

שינוי סטטוס משימה, הקצאה לסוכן אחר, או עדכון תיאור - ישירות מהכרטיס

⏻

כיבוי סוכנים

שליחת בקשת כיבוי מסודרת לסוכן - הסוכן יכול לאשר או לדחות אם עדיין עובד
