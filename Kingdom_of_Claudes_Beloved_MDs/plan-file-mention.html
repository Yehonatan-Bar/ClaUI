תוכנית: תיוג קבצים עם @

תוסף ClaUi - פיצ'ר חדש

מה הפיצ'ר?

כשהמשתמש מקליד @ בתיבת הצ'אט, מיד נפתח חלון קופץ שמציג קבצים מתוך הפרויקט. ככל שהמשתמש ממשיך להקליד - התוצאות מסוננות בזמן אמת. אפשר לנווט עם חצים ולבחור עם Enter, או ללחוץ בעכבר. הנתיב של הקובץ שנבחר מוכנס אוטומטית לתוך ההודעה.

למה צריך את זה?

היום, הדרך היחידה להכניס נתיב קובץ היא דרך כפתור + שפותח חלון בחירת קבצים של מערכת ההפעלה - תהליך איטי שדורש כמה לחיצות. הפיצ'ר החדש מאפשר למצוא ולהכניס קבצים ישירות תוך כדי כתיבה, בדיוק כמו תיוג ב-GitHub, Slack או Cursor.

איך זה עובד?

  1. המשתמש מקליד @ בתיבת הקלט
  2. מיד נפתח חלון קופץ מעל תיבת הקלט
  3. כל תו נוסף מסנן את רשימת הקבצים בזמן אמת
  4. חצי מקלדת (למעלה/למטה) מנווטים בין התוצאות
  5. Enter או לחיצת עכבר - מכניסים את נתיב הקובץ
  6. Escape סוגר את החלון הקופץ

שינויים בקוד

רכיב סוג שינוי תיאור
הגדרת הודעות עדכון הוספת שני סוגי הודעות חדשים - בקשת חיפוש ותוצאות חיפוש
מטפל הודעות (צד השרת) עדכון מטפל חדש שמחפש קבצים בפרויקט ומחזיר תוצאות
מאזין הודעות (צד הדפדפן) עדכון קליטת תוצאות חיפוש והעברתן לרכיב הקופץ
לוגיקת תיוג קבצים חדש לוגיקת זיהוי @, חיפוש עם השהייה, ניהול מצב הקופץ
חלון קופץ חדש רכיב ויזואלי שמציג רשימת קבצים תואמים
אזור הקלט עדכון שילוב הקופץ + יירוט מקשים כשהקופץ פתוח
עיצוב עדכון סגנונות CSS לחלון הקופץ בהתאמה לערכת הנושא של VS Code

התנהגות מקלדת

מקש כשהקופץ פתוח כשהקופץ סגור
חץ למעלה/למטה ניווט בתוצאות היסטוריית הודעות (כמו היום)
Enter בחירת קובץ שורה חדשה (כמו היום)
Escape סגירת הקופץ ביטול בקשה (כמו היום)
Ctrl+Enter שליחת הודעה שליחת הודעה (כמו היום)
