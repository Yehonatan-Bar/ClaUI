תוכנית: מתן שם אוטומטי לטאבים של סשנים

מטרה: כאשר המשתמש שולח את ההודעה הראשונה בסשן, המערכת תייצר אוטומטית שם קצר (1-3 מילים) לטאב, בשפת ההודעה (עברית או אנגלית).

איך זה עובד

1 המשתמש שולח הודעה ראשונה

כשהמשתמש שולח את ההודעה הראשונה בסשן חדש, המערכת מזהה שזו ההודעה הראשונה.

2 תהליך רקע מייצר שם

המערכת מפעילה תהליך קל ברקע שמבקש ממודל מהיר וזול (Haiku) לייצר שם של 1-3 מילים על בסיס תוכן ההודעה. השם נוצר בשפת ההודעה.

3 הטאב מתעדכן

כשהשם מוכן (בדרך כלל תוך מספר שניות), כותרת הטאב מתעדכנת אוטומטית. בינתיים, הטאב מציג את השם הרגיל ("Claude Mirror").

דוגמאות

"Add unit tests for the API" API Unit Tests
"עזור לי לכתוב README" כתיבת README

נקודות עיקריות

נושא פרטים
מתי מופעל רק בהודעה הראשונה של כל סשן חדש
שפת השם מותאמת לשפת ההודעה (עברית/אנגלית)
עלות מינימלית - שימוש במודל Haiku הזול והמהיר
חסימת המשתמש אין - התהליך רץ ברקע, לא מעכב שום דבר
טיפול בשגיאות אם יש בעיה, הטאב פשוט שומר על השם הרגיל - ללא הודעות שגיאה
ניתן לכיבוי הגדרה חדשה מאפשרת לכבות את הפיצ'ר
הערה: התהליך פועל ברקע ולא משפיע על חוויית המשתמש. אם משהו משתבש (הכלי לא נמצא, תם הזמן, תשובה לא תקינה), הטאב פשוט נשאר עם השם הרגיל בשקט.

רכיבים שמשתנים

רכיב שינוי
רכיב חדש: מחולל שמות רכיב חדש שמפעיל תהליך רקע קל ליצירת שם הסשן
חלון הצ'אט הוספת יכולת לעדכן את כותרת הטאב דינמית
גשר ההודעות זיהוי הודעה ראשונה והפעלת מחולל השמות
הגדרות הגדרה חדשה לכיבוי/הפעלה של הפיצ'ר

בדיקות

1 הודעה באנגלית -> שם טאב באנגלית
2 הודעה בעברית -> שם טאב בעברית
3 ניקוי סשן + הודעה חדשה -> שם חדש מותאם
4 כיבוי הפיצ'ר בהגדרות -> אין שינוי שם
