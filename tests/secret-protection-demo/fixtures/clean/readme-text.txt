Welcome to the Demo Application

This is a sample project for testing purposes. It demonstrates
best practices for TypeScript development with React and VS Code extensions.

Getting Started:
1. Clone the repository
2. Run npm install
3. Run npm run build
4. Open in VS Code

Architecture:
The application uses a modular architecture with separate concerns
for the extension host, webview UI, and shared utilities.
