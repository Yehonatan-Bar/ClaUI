01 Executive Summary

This report documents a controlled security experiment testing Particle Accelerator's ability to detect and redact sensitive data in AI coding agent command output. A test script containing 16 fake secrets across 14 distinct secret formats was executed. The system successfully identified and redacted 100% of all secrets before they could reach the AI model's context window.


When AI coding agents (Claude, Codex) execute shell commands, the raw output — which may contain API keys, tokens, passwords, and certificates — flows directly into the model's context. Particle Accelerator intercepts this output and scrubs all sensitive data before the AI ever sees it.

02 The Experiment

Methodology: A Bash script (pa-security-demo.sh) was created containing fake but format-accurate secrets matching all 14 regex patterns and environment variable patterns used by the SecretRedactor engine. The script was executed within a ClaUI session with Particle Accelerator enabled.


Two-phase comparison:

  • Without PA: Raw command output reaches the AI model, exposing all secrets
  • With PA: Intercepted output has all secrets replaced with [REDACTED]

06 Organizational Value

Defense in Depth

Adds an automated security layer between AI agents and sensitive data. Even if a developer accidentally commits secrets or they exist in environment variables, they are scrubbed before reaching any external AI provider.

Audit-Ready Control

Serves as evidence of proactive credential-exposure prevention as part of SOC 2 (Confidentiality) and ISO 27001 (Annex A, A.8.11 Data Masking / A.8.12 Data Leakage Prevention) compliance efforts. Adds a documented, automated DLP control to the organization's security posture.

Zero-Configuration Protection

14 secret patterns are detected automatically with no configuration needed. The system uses both regex pattern matching and environment variable value scanning for comprehensive coverage.

Fail-Closed Design

If the redaction engine encounters any error, the entire output is suppressed rather than allowing potentially sensitive data through. Security is never compromised by edge cases.

Conclusion

Particle Accelerator successfully detected and redacted 100% of secrets across all 14 tested patterns. The system provides automated, zero-configuration protection against secret leakage to AI providers, adding a critical security layer for organizations using AI coding agents in production environments.
